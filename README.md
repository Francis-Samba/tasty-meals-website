# Tasty Meals Website

## Image Setup

The hero section is ready to display your brand image. To complete the setup:

1. **Add your brand image**: 
   - Save your Tasty Meals logo/brand image as `hero-brand.jpg` in this folder
   - The image will automatically display in the hero section
   - Recommended dimensions: 400x400px or 500x500px

2. **File location**: 
   - Place the image file in: `c:\Users\francis.samba\Desktop\Tasty Meals\Website\hero-brand.jpg`

3. **The HTML will automatically reference it** in the hero section

## Local Development

Run the local server:
```bash
python -m http.server 8000
```

Visit: `http://localhost:8000`
